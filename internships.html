İlk stajımı Saat Teknoloji şirketinde front-end yazılımcı olarak react native dilinde 23 iş günü staj yapmış bulundum.
Stajım boyunca FreeCodeCamp üstünden Javascript sorularını çözdüm.
React-Native tutoriallarını çalışarak basit sayfa oluşturma görsel ekleme renk ekleme liste yapma sayfalar arası bağlantı kurmayı öğrendim.
Stajımın sonunda ise sportstv mobile uygulamasındaki hataları debugladım ve sportstv sayfası için basit bir blog sayfası oluşturdum.

İkinci stajımı ise adeo security şirketinde Blue Team Odaklı Siber Güvenlik Stajımı remote yapmış bulundum.
Stajımda sanal makine kurma, sanal makinelere windows 10, windows 2019 server, ve ubuntu kurmayı öğrendim.
Bu üç sanal makinelere splunk kurdum ve bunu ana bilgisayarımda açtığım splunk enterprise sayfasına bağlantısını kurdum.
Putty üstünden ubuntuya bağlantı sağladım.
Windows 19 server üstünden kullanıcı ekleme, çıkarma, çalışma saati düzenleme, kullanıcı loglarını görme ve site kaldırmayı öğrendim aynı zamanda siber saldırılar hakkında bilgi edindim.
